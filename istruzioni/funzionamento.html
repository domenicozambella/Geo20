Il materiale didattico della mia (DZ) parte del corso è archiviato in formato grezzo in questa repository 
github.com/domenicozambella/Geo20
Per utilizzarlo al meglio, bisogna disporre dell'applicazione Jupyter. Esiste un server Jupyter di ateneo

(Alternativamente è possibile installare tutto nel proprio computer. Le istruzioni sono riportare in calce.) Per utilizzare il server di ateneo bisogna atenticarsi. Le credenziali verranno spedite agli studenti del primo anno per email all'indirizzo istituzionale. Per scaricare il materiale didattico bisogna aprire un terminale (dal menu a tendina: New > Terminal) e dare il comando 
syncGeo20
Questa procedura crea la cartella Geo20 con tutto il materiale didattico. La stessa procedura va usata per aggiornare il materiale (che varierà spesso).

Il diario delle lezioni si trova nel file (da aprire all'interno di Jupyter) 
Geo20/lezioni/00_DiarioLezioni.ipynb
Questo stesso file contiene anche la bibiografia e l'agenda delle lezioni.

Per prima cosa quando si apre un notebook (ovvero un file con estensione .ipynb) bisogna eseguire tutte le celle. Dal menù a tendina: Cell > Run All.


Se le cose vanno storte, meglio ri-inizializzare il kernel. Dal menù a tendina: Kernel > Restart & Clear Output. Poi bisogna eseguire di nuovo tutte le celle.
